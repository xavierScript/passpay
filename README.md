PassPay is a passkey-powered Solana wallet with gasless transactions. Users create wallets using biometric authentication (FaceID, TouchID, Windows Hello) instead of seed phrases, and all transaction fees are covered by LazorKit Paymaster.

This repository contains the documentation and implementation for both web and mobile applications.
